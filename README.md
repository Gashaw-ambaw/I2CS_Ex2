# Ex2: Map2D BFS Pathfinding
> **Author:** Gashaw Ambaw | **ID:** 323361386

**Description:** An OOP Java project implementing 2D grid algorithms. It features a robust **BFS** implementation for shortest-path calculation and distance field generation.

**Key Features:**
* **BFS Logic:** Efficient implementation of `shortestPath` and `allDistance` supporting obstacle avoidance.
* **Coordinate Sync:** Handles the translation between Cartesian (x,y) and Matrix [row][col] indexing.
* **File IO:** Includes `save` and `load` functionality for map data with custom formatting.
* **Testing:** Fully verified by a comprehensive `JUnit` suite covering all edge cases.

**Usage:** Run the `MapTest` class (JUnit) to verify algorithmic correctness and logic.
